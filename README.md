# Sentinel AI – Autonomous Cybersecurity Threat Detection System

Sentinel AI is an AI-powered cybersecurity monitoring platform designed to detect and analyze cyber threats using machine learning and generative AI techniques.

## Features
- AI-based anomaly detection
- Real-time threat alerts
- Generative AI threat explanation
- Security monitoring dashboard

## Technology Stack
- Python
- Machine Learning
- Generative AI
- MongoDB
- Streamlit
